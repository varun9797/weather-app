# weather-app

# prerequisite 
Install Docker 

#### Bellow are the commands to run the project
# For Node Server
cd node-server &&
npm run docker 

# For React Client
cd ../react-client && 
npm run docker 

# For weather-stream-simulator
cd ../weather-stream-simulator &&
npm run docker 

# Lastly 
cd .. && 
docker compose up 

### Once Docker container start running open Url - http://localhost:5173/
